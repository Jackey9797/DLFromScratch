# DLFromScratch
本人是末流985计科的一名科研小白，进组之前通过这个项目（导师布置的）复习大一所学的深度学习知识。同时自己也算是想借此机会，正式把github的仓库功能给利用起来吧。希望以后能产出更有价值的东西。

本项目除了手写构建深度学习模型API、MLP/CNN/RNN几个经典模型之外，还融合了一些工程规范，和深度学习实验管理之类的工具和环境，以便以后写大工程时参考和复用。
